Description :
AthanTimes is a Qml application witch allow to get athan prayer times for a specific date & location.

Inputs : 
1- input a valid location on the first textField.
2- input a valid date with the corresponding format on the 2nd text field.

Output:
once the inputs are valid you can push the "get athan time" Button to get results as shown bellow
